Learning Management Platform (SaaS)

An award-winning B2B platform enabling a closed-loop learning ecosystem for tutoring centers. Transforming complex administrative and teaching workflows into actionable data insights.

Design lead

Background

Tutoring centers in Taiwan typically operate 1-2 weeks ahead of school curriculum, helping students preview and reinforce learning. However, most centers faced common operational challenges:

  • Fragmented Tools: Teachers juggled multiple platforms for content delivery, testing, and parent communication
  • Manual Workflows: Assigning tests and tracking completion required significant administrative effort
  • Limited Visibility: No systematic way to identify which students were falling behind or which concepts needed reinforcement
  • Communication Gaps: Parents lacked transparency into their child's learning progress

To address these challenges, we designed this platform, a unified learning management platform that streamlines the entire teaching workflow while providing actionable insights for teachers and transparency for parents.

Key Challenge

  • Building a comprehensive learning management system from scratch for tutoring centers with diverse operational needs
  • Designing for three distinct user roles (admin, teacher, student/parent) with different workflows and goals
  • Creating data visualization dashboards that help teachers quickly identify struggling students
  • Integrating multiple learning products (e-books, tests, oral practice) into one unified platform

Solution

  • Established a closed-loop learning cycle: Preview ‚Üí Class ‚Üí Test ‚Üí Diagnosis ‚Üí Remedial Practice
  • Designed role-based interfaces for institution admins, teachers, and students/parents
  • Built comprehensive analytics dashboards including learning progress tracking and knowledge gap analysis

The Goal

Creating a Closed-Loop Learning Cycle

Based on our discovery, we focused on building a system that supports the complete tutoring workflow:

  1. One-Stop Platform: Build a comprehensive Learning Management System (LMS) that integrates administrative management, teaching activities, resource management, and learning feedback into a unified platform
  2. Operational Efficiency: Provide real-time activity tracking so teachers can monitor student progress and task completion at a glance
  3. Data-Driven Teaching: Deliver analytics dashboards (Learning Analytics, Comprehensive Reports) that reveal student mastery levels across knowledge points, enabling targeted intervention
  4. Closed-Loop Learning: Support personalized remediation through wrong-answer practice, reinforcing weak areas and completing the learning cycle
Strategy & Solution

From Fragmented Tools to Integrated Learning Cycle

We adopted a cycle-based design strategy‚Äîtreating the learning process as a continuous loop rather than discrete activities‚Äîcreating a seamless experience for teachers to manage and optimize student learning.

üéØ Challenge 1: No Centralized View of Student Progress

Problem

  • Teachers had no single dashboard to monitor class progress
  • Identifying struggling students required checking multiple sources

Solution

  • Instant class health check: Comprehensive report dashboard displays average score rate, assessment count, and auto-identified weakest knowledge points in one view
  • Data-driven teaching decisions: Knowledge point error rate ranking and cognitive dimension radar chart reveal exactly what to re-teach and whether students struggle with recall or deeper comprehension
üéØ Challenge 2: Time-Consuming Test Assignment

Problem

  • Manually scheduling and assigning tests for each class took significant time
  • No systematic way to align with school curriculum pacing

Solution

  • Designed Auto-scheduling system that dispatches content 1-2 weeks ahead of school curriculum
  • Created Activity Management interface for bulk assignment and tracking
  • Built Assignment Status Dashboard showing real-time completion rates
üéØ Challenge 3: No Systematic Remediation

Problem

  • Time-consuming manual review: Teachers spent significant time reviewing each test paper to identify individual student errors
  • Inefficient remediation: Without targeted mechanisms, teachers could only assign "one-size-fits-all" retests to the entire class
  • Student frustration: Students were forced to repeat content they had already mastered, wasting time and reducing learning effectiveness

Solution

  • Spot learning gaps at a glance: Knowledge point mastery heatmap with color-coded cells lets teachers instantly identify high-error-rate concepts for the class or individual students
  • One-click targeted assignment: Teachers can select weak knowledge points and struggling students, then dispatch personalized remedial practice directly from the analytics page
  • Automated personalized review: System auto-generates individualized review paths‚Äîstudents only practice concepts they haven't mastered, with correctly answered questions automatically removed from future assignments

Project Reflection

Turning Complexity into Clarity

Beyond the metrics, this project taught me that designing for B2B isn't just about interfaces‚Äîit's about deeply understanding operational workflows before proposing solutions.

From Features to Systems

Designing this platform taught me that effective B2B products aren't just about features; they are about orchestrating complex workflows. By transforming fragmented manual tasks into a unified "Closed-Loop" system, I learned how to identify operational bottlenecks and solve them through design.

Empowering Decisions through Data Visibility

The biggest challenge was making invisible learning gaps visible. I realized that for a dashboard to be truly useful, it must move beyond displaying data to triggering actions. Designing these analytics interfaces honed my ability to prioritize information hierarchy for different roles (Admins vs. Teachers), ensuring users can instantly detect anomalies and take action.
